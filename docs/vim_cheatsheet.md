# Vim Cheatsheet

## Modes
- `i`: Insert mode.
- `Esc`: Normal mode.
- `:w`: Save.
- `:q`: Quit.
- `:wq`: Save and quit.

## Navigation
- `h`: Left.
- `j`: Down.
- `k`: Up.
- `l`: Right.
- `gg`: Go to top.
- `G`: Go to bottom.

## Editing
- `dd`: Delete line.
- `yy`: Yank (copy) line.
- `p`: Paste.
- `u`: Undo.
