# Regex Cheatsheet

## Anchors
- `^`: Start of string.
- `$`: End of string.
- `\b`: Word boundary.

## Character Classes
- `.`: Any character.
- `\d`: Digit.
- `\w`: Word character.
- `\s`: Whitespace.
- `[abc]`: Any of a, b, or c.
- `[^abc]`: Not a, b, or c.

## Quantifiers
- `*`: 0 or more.
- `+`: 1 or more.
- `?`: 0 or 1.
- `{n}`: Exactly n times.
- `{n,}`: n or more times.
- `{n,m}`: Between n and m times.
