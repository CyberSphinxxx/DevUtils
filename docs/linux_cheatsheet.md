# Linux Cheatsheet

## File Operations
- `ls`: List directory.
- `cd`: Change directory.
- `cp`: Copy file.
- `mv`: Move file.
- `rm`: Remove file.
- `mkdir`: Make directory.

## Permissions
- `chmod`: Change mode.
- `chown`: Change owner.

## Process Management
- `ps`: Process status.
- `top`: Display processes.
- `kill`: Terminate process.
